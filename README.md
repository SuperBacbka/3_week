# 3_week
Программный модуль предназначен для автоматизации учета заявок на ремонт климатического оборудования, 
назначения исполнителей, контроля сроков выполнения, ведения истории изменений статусов, 
организации контроля качества и сбора отзывов клиентов.
Состав программного обеспечения:
app.py — пользовательский интерфейс (Streamlit)
database.py — модуль работы с базой данных SQLite
import_resources.py — модуль загрузки данных из CSV-файлов
test_database.py — модуль функционального тестирования (pytest)
База данных service_center.db, service_center_import.db
Условия эксплуатации

Операционная система: Windows 10 / Windows 11

Язык программирования: Python версии 3.10 и выше

СУБД: SQLite 3

Среда разработки: PyCharm

Дополнительные библиотеки:

streamlit

pandas

plotly

qrcode (опционально)

pytest


Программный модуль использует:

локальную файловую БД SQLite;

библиотеку Streamlit для построения пользовательского интерфейса;

библиотеку pytest для функционального тестирования;

Google Forms для сбора отзывов клиентов.
Порядок работы

В системе реализованы следующие роли:

Администратор — управление пользователями и заявками;

Специалист (мастер) — выполнение ремонта, изменение статусов заявок, отправка запроса «Нужна помощь»;

Менеджер по качеству — контроль сроков, привлечение мастеров, продление сроков с согласованием клиента.

Контроль качества:
После завершения ремонта в карточке заявки отображается QR-код для перехода на форму оценки качества.
